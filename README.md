
# 🧑‍💻 CRUD de Usuários (Frontend)

Projeto de CRUD de usuários desenvolvido com HTML, CSS e JavaScript puro, integrado a uma API REST em Node.js.

## 🚀 Funcionalidades

- Criar usuários
- Listar usuários
- Buscar usuários por nome
- Editar usuários
- Excluir usuários

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Fetch API
